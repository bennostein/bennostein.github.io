About the Application

This app extends Google Sheets, adding some simple filtering and sorting utilities in order to support the needs of multiple simultaneous users in our large spreadsheets.
It provides an app script that applies specific sorts and filters to Google Sheets, and allows users to quickly view the data they need or to reset the filters so that other users can view the full data set.

Permissions and Privacy

The permissions requested by the application through Google are the minimum required for the desired filtering and sorting capabilities, and the application does not access any user data outside of the Movement Voter Project-internal spreadsheets it operates on.
In particular, MVP SS Tools does not read, modify, or delete any user-owned spreadsheets.
